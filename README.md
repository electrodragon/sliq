# Sliq

Sliq is a lightweight Linux presentation viewer powered by HTML, GSAP animations, and your default browser.

It instantly turns any folder of images into a fullscreen animated presentation.

Perfect for:

- Project demos
- College presentations
- Portfolio showcases
- Offline slides
- Fast image decks
- Developer workflows

---

# Features

- ⚡ Instant presentation from any folder
- 🖼️ Automatic image discovery
- 🎞️ GSAP-powered animations
- ⌨️ Keyboard navigation
- 🖥️ Fullscreen support
- 🐧 Linux-first utility
- 🌐 Browser-powered rendering
- 📂 Supports JPG, PNG, WEBP, GIF, SVG, AVIF and more
