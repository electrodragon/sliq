Sliq

A minimal, fast presentation viewer for the terminal-first generation.

install.sh
$ bash <(curl -s -H "Cache-Control: no-cache" 'https://raw.githubusercontent.com/electrodragon/sliq/main/install.sh')
/ Usage

Run it anywhere.

Point Sliq at any directory of images and it builds a presentation instantly. No config, no setup.

~/terminal
$ sliq                          # current directory
$ sliq .                        # explicit cwd
$ sliq ../presentation          # relative path
$ sliq ~/Pictures/slides        # absolute path
/ Features
